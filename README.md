# Pokedex in [SvelteKit](https://kit.svelte.dev/) as classic full stack application

- used for Apple Music & NY Times

## Quickstart

```
npm create skeleton-app@latest my-skeleton-app
	- Enable SvelteKit's Typescript syntax (recommended)
cd my-skeleton-app
```
